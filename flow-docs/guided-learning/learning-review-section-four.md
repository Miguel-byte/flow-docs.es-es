# <a name="review-the-fourth-section"></a>Repaso de la cuarta sección
Ahora revisaremos lo que hemos aprendido en esta sección.

**Microsoft Flow** tiene varias características para **usuarios de empresa**: 

* Un **entorno** es un espacio virtual utilizado para almacenar, administrar y compartir aplicaciones, flujos y datos empresariales dentro de Common Data Service.  Los entornos están geolocalizados, por lo que todas las aplicaciones y los datos almacenados en la base de datos de un entorno también lo están.
* Con Microsoft Flow, puede crear fácilmente directivas de **prevención de pérdida de datos** (DLP) para controlar de forma estricta con qué servicios al consumidor pueden compartirse sus datos empresariales cuando los usuarios creen flujos.
